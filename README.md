# Rock Paper Scissors Game in C 
 
A simple Rock Paper Scissors game built using the C programming language. 
 
## Features 
- User vs Computer gameplay 
- Random computer choices 
- Input validation 
- Replay option 
 
## Technologies Used 
- C Programming Language 
 
## How to Run 
1. Compile the code using GCC 
2. Run the executable 
 
Example: 
gcc main.c -o game 
.\game 
 
## Game Rules 
- Rock beats Scissors 
- Scissors beats Paper 
- Paper beats Rock
